Консольное приложение для управления личными финансами с системой бюджетирования и категоризации расходов.

Пример работы приложения:
https://www.dropbox.com/scl/fi/dvtcdv3ma5c9novllkafj/unknown_2025.10.08-19.57.mp4?rlkey=z7npw728rb75c0uzb2sg58gzz&st=s8dgettv&dl=0

✨ Возможности
- Система аутентификации - регистрация и вход пользователей
- Учет доходов и расходов - по категориям с автоматическим расчетом баланса
- Бюджетирование - установка лимитов по категориям с предупреждениями о превышении
- Статистика - детальная аналитика финансовой деятельности
- Docker-поддержка - запуск на любой системе без установки Java

🚀 Быстрый старт

Предварительные требования
Docker Desktop установленный и запущенный

Запуск приложения:
- Клонируйте репозиторий (или используйте существующую папку с проектом)
- Откройте терминал в папке проекта
- Запустите команду:
- docker-compose up --build

Приложение запустится и покажет главное меню

📖 Руководство пользователя

Важно: Тестирование только через терминал!

Приложение требует интерактивного ввода. Используйте:
- Windows: PowerShell или Command Prompt
- Mac/Linux: Terminal
- Не используйте Docker Desktop GUI для тестирования функций приложения
