🎯 Назначение проекта
finance-app - это консольное приложение, разработанное для эффективного учета личных финансов. Оно позволяет пользователям:
1. Отслеживать доходы и расходы по категориям
2. Устанавливать бюджетные ограничения
3. Анализировать финансовую статистику
4. Контролировать соблюдение бюджета
Решаемая проблема: Помощь пользователям в планировании бюджета и предотвращении неконтролируемых трат через систему предупреждений.

✨ Ключевые возможности
🔐 Система аутентификации
- Регистрация новых пользователей
- Авторизация с проверкой учетных данных
- Безопасный выход из системы

💰 Управление финансами
Внесение доходов с указанием категории
Фиксация расходов с автоматической проверкой бюджета
Установка бюджетов по категориям
Предупреждения при превышении бюджета

📊 Аналитика и отчетность
- Текущий баланс (расчетный)
- Общие суммы доходов и расходов
- Остаток бюджета по каждой категории
- Детальная статистика по всем операциям
